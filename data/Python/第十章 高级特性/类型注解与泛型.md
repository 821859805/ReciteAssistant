**基本类型注解（Python 3.5+）：**
```python
# 变量注解
name: str = "Alice"
age: int = 25
scores: list[float] = [98.5, 87.0]

# 函数注解
def greet(name: str, times: int = 1) -> str:
    return (f"Hello, {name}! " * times).strip()
```

**typing 模块常用类型：**
```python
from typing import (
    Optional, Union, Any,
    List, Dict, Tuple, Set,
    Callable, Iterator, Generator,
    TypeVar, Generic
)

# Optional —— 可能为 None
def find(name: str) -> Optional[int]:  # 等价于 int | None
    ...

# Union —— 多种类型
def process(data: Union[str, bytes]) -> None:  # 等价于 str | bytes
    ...

# Python 3.10+ 新语法
def process(data: str | bytes) -> None:
    ...

# Callable —— 可调用对象
def apply(func: Callable[[int, int], int], a: int, b: int) -> int:
    return func(a, b)

# 嵌套类型
Matrix = list[list[float]]          # Python 3.9+
UserInfo = dict[str, str | int]     # Python 3.10+
```

**TypeVar 与泛型：**
```python
from typing import TypeVar, Generic

T = TypeVar('T')

def first(items: list[T]) -> T:
    return items[0]

# 泛型类
class Stack(Generic[T]):
    def __init__(self) -> None:
        self._items: list[T] = []
    
    def push(self, item: T) -> None:
        self._items.append(item)
    
    def pop(self) -> T:
        return self._items.pop()

stack: Stack[int] = Stack()
stack.push(42)
```

**Python 3.12+ 新语法：**
```python
# 更简洁的泛型语法
def first[T](items: list[T]) -> T:
    return items[0]

class Stack[T]:
    def __init__(self) -> None:
        self._items: list[T] = []

# type 关键字定义类型别名
type Vector = list[float]
type Matrix = list[Vector]
```
