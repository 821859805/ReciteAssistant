**1. 位置参数（Positional）：**
```python
def greet(name, greeting):
    print(f"{greeting}, {name}!")

greet("Alice", "Hello")        # 按位置传递
greet(greeting="Hi", name="Bob")  # 按关键字传递
```

**2. 默认参数（Default）：**
```python
def greet(name, greeting="Hello"):
    print(f"{greeting}, {name}!")

greet("Alice")           # Hello, Alice!
greet("Alice", "Hi")     # Hi, Alice!

# ⚠️ 默认值陷阱：可变对象作为默认值
def append_to(element, lst=[]):   # 危险！
    lst.append(element)
    return lst

append_to(1)   # [1]
append_to(2)   # [1, 2]  ——不是 [2]！默认列表被共享

# 正确做法：使用 None
def append_to(element, lst=None):
    if lst is None:
        lst = []
    lst.append(element)
    return lst
```

**3. 可变位置参数（*args）：**
```python
def total(*args):
    print(type(args))    # <class 'tuple'>
    return sum(args)

total(1, 2, 3)          # 6
total(1, 2, 3, 4, 5)    # 15
```

**4. 可变关键字参数（**kwargs）：**
```python
def show_info(**kwargs):
    print(type(kwargs))  # <class 'dict'>
    for k, v in kwargs.items():
        print(f"{k}: {v}")

show_info(name="Alice", age=25)
```

**5. 仅限位置参数（/）和仅限关键字参数（*）：**
```python
# Python 3.8+
def func(pos_only, /, normal, *, kw_only):
    pass

func(1, 2, kw_only=3)       # OK
func(1, normal=2, kw_only=3) # OK
# func(pos_only=1, ...)      # TypeError! pos_only 只能按位置传
# func(1, 2, 3)              # TypeError! kw_only 只能按关键字传
```

**参数顺序规则：**
`位置参数 → /（仅位置参数标记） → 普通参数 → *args → 仅关键字参数 → **kwargs`
