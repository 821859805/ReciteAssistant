**基本语法：**
```python
def greet(name):
    """问候函数"""
    return f"Hello, {name}!"

# 调用
result = greet("Alice")
print(result)  # Hello, Alice!
```

**函数也是对象（第一等公民）：**
```python
# 函数可以赋值给变量
say_hi = greet
say_hi("Bob")       # Hello, Bob!

# 函数可以作为参数传递
def apply(func, value):
    return func(value)
apply(greet, "Charlie")  # Hello, Charlie!

# 函数可以存入数据结构
funcs = [str.upper, str.lower, str.title]
for f in funcs:
    print(f("hello world"))
```

**返回值：**
```python
# 无 return 或 return 后无值，返回 None
def do_nothing():
    pass
result = do_nothing()  # None

# 返回多个值（实际是返回元组）
def divmod_custom(a, b):
    return a // b, a % b

quotient, remainder = divmod_custom(10, 3)  # 3, 1

# 提前返回
def find_first_negative(lst):
    for item in lst:
        if item < 0:
            return item
    return None
```

**函数注解（Type Hints）：**
```python
def add(a: int, b: int) -> int:
    return a + b

# 复杂类型注解
from typing import Optional, Union
def find(name: str) -> Optional[int]:
    ...
def process(data: Union[str, list]) -> None:
    ...

# Python 3.10+
def process(data: str | list) -> None:
    ...
```
