**闭包**是一个函数，它记住了创建时所在作用域的变量，即使该作用域已经结束。

**基本形式：**
```python
def make_multiplier(n):
    def multiplier(x):
        return x * n    # n 来自外层函数
    return multiplier    # 返回内部函数

double = make_multiplier(2)
triple = make_multiplier(3)
print(double(5))   # 10
print(triple(5))   # 15
```

**闭包的条件：**
1. 有嵌套函数
2. 内部函数引用了外部函数的变量
3. 外部函数返回内部函数

**查看闭包的自由变量：**
```python
print(double.__closure__)           # (<cell ...>,)
print(double.__closure__[0].cell_contents)  # 2
```

**闭包中修改外部变量：**
```python
def make_counter():
    count = 0
    def counter():
        nonlocal count   # 必须使用 nonlocal
        count += 1
        return count
    return counter

c = make_counter()
print(c())  # 1
print(c())  # 2
print(c())  # 3
```

**经典陷阱——循环中的闭包：**
```python
# 错误示例
funcs = []
for i in range(3):
    funcs.append(lambda: i)
print([f() for f in funcs])  # [2, 2, 2]  ——全是2！

# 原因：lambda 引用的是变量 i，循环结束后 i=2

# 解决方案1：默认参数捕获
funcs = []
for i in range(3):
    funcs.append(lambda i=i: i)
print([f() for f in funcs])  # [0, 1, 2]

# 解决方案2：使用工厂函数
def make_func(i):
    return lambda: i
funcs = [make_func(i) for i in range(3)]
```
