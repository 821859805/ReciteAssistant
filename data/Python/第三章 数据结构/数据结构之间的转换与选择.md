**相互转换：**
```python
# list <-> tuple
list((1, 2, 3))      # [1, 2, 3]
tuple([1, 2, 3])     # (1, 2, 3)

# list <-> set（去重）
list({1, 2, 3})      # [1, 2, 3]（顺序不定）
set([1, 2, 2, 3])    # {1, 2, 3}

# dict -> list
d = {"a": 1, "b": 2}
list(d)              # ['a', 'b']（键列表）
list(d.values())     # [1, 2]（值列表）
list(d.items())      # [('a', 1), ('b', 2)]

# list of tuples -> dict
dict([("a", 1), ("b", 2)])  # {'a': 1, 'b': 2}

# str <-> list
list("hello")        # ['h', 'e', 'l', 'l', 'o']
"".join(['h', 'e'])  # "he"
```

**如何选择数据结构：**

| 场景 | 推荐 | 原因 |
|------|------|------|
| 有序、可变的集合 | list | 最通用的序列 |
| 不可变的固定数据 | tuple | 安全、可哈希、性能更好 |
| 键值映射 | dict | O(1) 查找 |
| 去重、集合运算 | set | 自动去重、O(1) 成员检测 |
| FIFO 队列 | collections.deque | 两端操作 O(1) |
| 计数器 | collections.Counter | 统计元素频率 |
| 有序字典 | dict（3.7+） | 保持插入顺序 |
| 默认值字典 | collections.defaultdict | 自动初始化 |

**时间复杂度对比：**
| 操作 | list | dict | set |
|------|------|------|-----|
| 查找 | O(n) | O(1) | O(1) |
| 插入 | O(1)尾/O(n)头 | O(1) | O(1) |
| 删除 | O(n) | O(1) | O(1) |
