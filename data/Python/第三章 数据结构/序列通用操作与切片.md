**通用序列操作（适用于 str, list, tuple）：**
```python
s = [10, 20, 30, 40, 50]

# 索引
s[0]      # 10
s[-1]     # 50

# 切片 [start:stop:step]
s[1:3]    # [20, 30]
s[:3]     # [10, 20, 30]
s[2:]     # [30, 40, 50]
s[::2]    # [10, 30, 50]
s[::-1]   # [50, 40, 30, 20, 10]

# 拼接与重复
[1, 2] + [3, 4]   # [1, 2, 3, 4]
[0] * 5            # [0, 0, 0, 0, 0]

# 成员判断
30 in s            # True

# 长度、最值、求和
len(s)    # 5
min(s)    # 10
max(s)    # 50
sum(s)    # 150
```

**切片对象（slice）：**
```python
# 切片可以作为对象传递
sl = slice(1, 4)
s[sl]              # 等价于 s[1:4]

# 切片赋值（仅 list）
lst = [0, 1, 2, 3, 4]
lst[1:3] = [10, 20, 30]   # [0, 10, 20, 30, 3, 4]
lst[1:4] = []              # [0, 3, 4]（删除元素）
```

**解包操作（*运算符）：**
```python
# 基本解包
a, b, c = [1, 2, 3]

# 星号解包
first, *rest = [1, 2, 3, 4, 5]
# first = 1, rest = [2, 3, 4, 5]

*init, last = [1, 2, 3, 4, 5]
# init = [1, 2, 3, 4], last = 5

a, *middle, z = [1, 2, 3, 4, 5]
# a = 1, middle = [2, 3, 4], z = 5

# 函数调用中解包
def add(a, b, c):
    return a + b + c
nums = [1, 2, 3]
add(*nums)          # 等价于 add(1, 2, 3)
```
