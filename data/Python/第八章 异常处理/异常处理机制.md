**基本语法：**
```python
try:
    result = 10 / 0
except ZeroDivisionError:
    print("不能除以零")
except (TypeError, ValueError) as e:
    print(f"类型或值错误: {e}")
except Exception as e:
    print(f"其他错误: {e}")
else:
    # try 块没有异常时执行
    print(f"结果: {result}")
finally:
    # 无论是否异常都执行（常用于资源清理）
    print("执行完毕")
```

**执行流程：**
1. 执行 `try` 块
2. 如有异常 → 匹配 `except`（从上到下，第一个匹配的）
3. 如无异常 → 执行 `else`
4. 无论如何 → 执行 `finally`

**主动抛出异常：**
```python
def set_age(age):
    if not isinstance(age, int):
        raise TypeError("age 必须是整数")
    if age < 0 or age > 150:
        raise ValueError(f"无效的年龄: {age}")
    return age

# 重新抛出异常
try:
    set_age(-1)
except ValueError:
    print("记录错误日志")
    raise   # 重新抛出当前异常
```
