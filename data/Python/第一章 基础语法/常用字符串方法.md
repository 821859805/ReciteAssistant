**查找与替换：**
```python
s = "Hello, World!"

s.find("World")        # 7（返回首次出现的索引，未找到返回-1）
s.index("World")       # 7（未找到抛出 ValueError）
s.count("l")           # 3
s.replace("World", "Python")  # "Hello, Python!"
s.startswith("Hello")  # True
s.endswith("!")        # True
"World" in s           # True（成员运算符）
```

**大小写转换：**
```python
"hello".upper()        # "HELLO"
"HELLO".lower()        # "hello"
"hello world".title()  # "Hello World"
"hello world".capitalize()  # "Hello world"
"Hello".swapcase()     # "hELLO"
```

**去除空白与填充：**
```python
"  hello  ".strip()    # "hello"（去除两端空白）
"  hello  ".lstrip()   # "hello  "（去除左端）
"  hello  ".rstrip()   # "  hello"（去除右端）
"hello".center(20, '-')  # "-------hello--------"
"42".zfill(5)          # "00042"（零填充）
```

**分割与连接：**
```python
"a,b,c".split(",")       # ['a', 'b', 'c']
"hello world".split()     # ['hello', 'world']（按空白分割）
"a\nb\nc".splitlines()    # ['a', 'b', 'c']
",".join(["a", "b", "c"]) # "a,b,c"
```

**判断类方法：**
```python
"123".isdigit()     # True（全是数字）
"abc".isalpha()     # True（全是字母）
"abc123".isalnum()  # True（字母或数字）
"  ".isspace()      # True（全是空白）
```
