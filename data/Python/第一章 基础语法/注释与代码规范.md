**注释方式：**
```python
# 单行注释

"""
多行注释（实际是多行字符串，未赋值时会被忽略）
常用于文档字符串
"""

def add(a, b):
    """计算两数之和。
    
    Args:
        a: 第一个数
        b: 第二个数
    
    Returns:
        两数之和
    """
    return a + b

# 查看文档字符串
help(add)
print(add.__doc__)
```

**PEP 8 代码规范要点：**
- **缩进**：使用4个空格，不用 Tab
- **行宽**：每行不超过79字符（docstring/注释72字符）
- **空行**：顶层定义之间空两行，方法之间空一行
- **导入**：每行一个 import，分组排列（标准库 → 第三方 → 本地）
- **命名规范**：
  - 变量/函数：`snake_case`
  - 类名：`PascalCase`
  - 常量：`ALL_CAPS`
  - 私有：`_private`，名称修饰：`__mangled`
- **空格**：运算符两侧各一个空格，逗号后一个空格
- **比较**：使用 `is` / `is not` 比较 `None`

**类型注解（Python 3.5+）：**
```python
def greet(name: str) -> str:
    return f"Hello, {name}"

age: int = 25
scores: list[float] = [98.5, 87.0]
```
类型注解不强制执行，仅供 IDE 和类型检查工具（如 mypy）使用。
