**整数 int：**
```python
# Python 3 的整数精度无限制
big_num = 10 ** 100   # googol

# 不同进制表示
binary = 0b1010       # 二进制 -> 10
octal = 0o17          # 八进制 -> 15
hexadecimal = 0xFF    # 十六进制 -> 255

# 数字分隔符（提高可读性）
population = 1_000_000_000
```

**浮点数 float：**
```python
pi = 3.14159
sci = 1.5e-3         # 科学计数法 -> 0.0015

# 浮点数精度问题（IEEE 754）
0.1 + 0.2            # 0.30000000000000004
# 解决：使用 decimal 模块
from decimal import Decimal
Decimal('0.1') + Decimal('0.2')  # Decimal('0.3')

# 特殊值
float('inf')         # 正无穷
float('-inf')        # 负无穷
float('nan')         # 非数字
```

**复数 complex：**
```python
z = 3 + 4j
z.real    # 3.0
z.imag    # 4.0
abs(z)    # 5.0（模）
```

**常用数学运算：**
```python
10 / 3    # 3.3333...（真除法）
10 // 3   # 3（整除，向下取整）
10 % 3    # 1（取余）
2 ** 10   # 1024（幂运算）
abs(-5)   # 5
round(3.7)  # 4
divmod(10, 3)  # (3, 1) —— 同时返回商和余数
```
