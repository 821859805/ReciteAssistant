**变量命名规则：**
- 由字母、数字、下划线组成，不能以数字开头
- 区分大小写（`name` 和 `Name` 是不同变量）
- 不能使用 Python 关键字（如 `if`, `for`, `class` 等）
- 推荐使用 snake_case 风格（如 `my_variable`）

**赋值方式：**
```python
# 基本赋值
x = 10
name = "Alice"

# 多重赋值
a, b, c = 1, 2, 3

# 链式赋值
x = y = z = 0

# 交换变量（Python 特有的优雅写法）
a, b = b, a

# 增量赋值
x += 1    # 等价于 x = x + 1
x -= 2
x *= 3
x /= 4
x //= 2   # 整除赋值
x **= 2   # 幂赋值
x %= 3    # 取余赋值
```

**变量本质：**
Python 中变量是对象的引用（标签），而非内存盒子。赋值操作是将变量名绑定到对象上。
```python
a = [1, 2, 3]
b = a          # b 和 a 指向同一个列表对象
b.append(4)
print(a)       # [1, 2, 3, 4]  —— a 也被改变了

# 使用 id() 查看对象内存地址
print(id(a) == id(b))  # True
```
