Python 的内置数据类型分为以下几类：

**数值类型：**
| 类型 | 说明 | 示例 |
|------|------|------|
| `int` | 整数，任意精度 | `42`, `-7`, `0xff` |
| `float` | 浮点数（双精度） | `3.14`, `1e-5` |
| `complex` | 复数 | `3+4j` |
| `bool` | 布尔值 | `True`, `False` |

**序列类型：**
| 类型 | 说明 | 特点 |
|------|------|------|
| `str` | 字符串 | 不可变，有序 |
| `list` | 列表 | 可变，有序 |
| `tuple` | 元组 | 不可变，有序 |

**映射类型：**
- `dict`：字典，键值对集合，键唯一

**集合类型：**
- `set`：可变集合，元素唯一
- `frozenset`：不可变集合

**其他类型：**
- `NoneType`：只有一个值 `None`，表示空
- `bytes` / `bytearray`：字节序列

**类型检查：**
```python
type(42)           # <class 'int'>
isinstance(42, int)  # True（推荐方式，支持继承检查）
isinstance(True, int)  # True，因为 bool 是 int 的子类
```
