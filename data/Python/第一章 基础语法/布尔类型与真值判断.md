**布尔值：**
```python
True   # 真
False  # 假

# bool 是 int 的子类
isinstance(True, int)   # True
True + True             # 2
True * 10               # 10
```

**假值（Falsy）：**
以下值在布尔上下文中为 `False`：
```python
bool(False)     # False
bool(None)      # False
bool(0)         # False（包括 0.0, 0j）
bool("")        # False（空字符串）
bool([])        # False（空列表）
bool(())        # False（空元组）
bool({})        # False（空字典）
bool(set())     # False（空集合）
```

**其他所有值都是真值（Truthy）。**

**逻辑运算符：**
```python
# and: 短路与 —— 第一个假值或最后一个值
0 and 1       # 0
1 and 2       # 2
1 and 2 and 3 # 3

# or: 短路或 —— 第一个真值或最后一个值
0 or 1        # 1
0 or "" or [] # []

# not: 取反
not True      # False
not 0         # True
not []        # True
```

**短路求值的妙用：**
```python
# 设置默认值
name = input_name or "Anonymous"

# 条件执行
debug and print("调试信息")  # 仅 debug 为真时打印
```
