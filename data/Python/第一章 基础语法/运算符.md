**算术运算符：**
| 运算符 | 说明 | 示例 |
|--------|------|------|
| `+` | 加 | `3 + 2 → 5` |
| `-` | 减 | `3 - 2 → 1` |
| `*` | 乘 | `3 * 2 → 6` |
| `/` | 除（真除法） | `7 / 2 → 3.5` |
| `//` | 整除 | `7 // 2 → 3` |
| `%` | 取余 | `7 % 2 → 1` |
| `**` | 幂 | `2 ** 3 → 8` |

**比较运算符：**
```python
3 == 3.0    # True（值相等）
3 is 3.0    # False（不是同一对象）
1 < 2 < 3   # True（链式比较）
1 < 2 and 2 < 3  # 等价写法
```

**位运算符：**
```python
a = 0b1010  # 10
b = 0b1100  # 12
a & b       # 0b1000 -> 8   （按位与）
a | b       # 0b1110 -> 14  （按位或）
a ^ b       # 0b0110 -> 6   （按位异或）
~a          # -11            （按位取反）
a << 2      # 0b101000 -> 40 （左移）
a >> 1      # 0b0101 -> 5    （右移）
```

**身份运算符 is：**
```python
a = [1, 2]
b = [1, 2]
a == b      # True（值相等）
a is b      # False（不是同一对象）

# 对 None 使用 is 比较
x = None
x is None   # True（推荐写法）
x == None   # True（不推荐）
```

**运算符优先级（从高到低）：**
`**` → `~` → `* / // %` → `+ -` → `<< >>` → `&` → `^` → `|` → `比较` → `not` → `and` → `or`
