**print() 函数：**
```python
# 基本输出
print("Hello, World!")

# 多个值（默认空格分隔）
print("a", "b", "c")       # a b c

# 自定义分隔符
print("a", "b", "c", sep="-")  # a-b-c

# 自定义结尾（默认换行）
print("hello", end=" ")
print("world")              # hello world（同一行）

# 输出到文件
with open("log.txt", "w") as f:
    print("日志内容", file=f)
```

**input() 函数：**
```python
# 基本输入（返回值始终是字符串）
name = input("请输入姓名: ")

# 输入数字需要转换
age = int(input("请输入年龄: "))
height = float(input("请输入身高: "))

# 一行输入多个值
a, b = input("输入两个数: ").split()
a, b = int(a), int(b)

# 更优雅的方式
nums = list(map(int, input("输入多个数: ").split()))
```

**格式化输出控制：**
```python
# 对齐与宽度
f"{'左对齐':<10}|"     # '左对齐       |'
f"{'右对齐':>10}|"     # '       右对齐|'
f"{'居中':^10}|"       # '    居中    |'

# 数字格式化
f"{1234567:,}"          # '1,234,567'（千位分隔）
f"{0.618:.2%}"          # '61.80%'（百分比）
f"{255:#x}"             # '0xff'（十六进制带前缀）
f"{42:#b}"              # '0b101010'（二进制带前缀）
```
