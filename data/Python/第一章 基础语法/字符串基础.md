**字符串定义：**
```python
s1 = 'hello'          # 单引号
s2 = "hello"          # 双引号（等价）
s3 = '''多行          # 三引号（多行字符串）
字符串'''
s4 = r"C:\Users\test"  # 原始字符串，反斜杠不转义
s5 = f"名字是{name}"    # f-string 格式化（Python 3.6+）
```

**转义字符：**
| 转义字符 | 含义 |
|----------|------|
| `\n` | 换行 |
| `\t` | 制表符 |
| `\\` | 反斜杠 |
| `\'` | 单引号 |
| `\"` | 双引号 |

**字符串索引与切片：**
```python
s = "Python"
s[0]      # 'P'（正向索引，从0开始）
s[-1]     # 'n'（负向索引，从-1开始）
s[0:3]    # 'Pyt'（切片 [start:stop)）
s[::2]    # 'Pto'（步长为2）
s[::-1]   # 'nohtyP'（反转字符串）
```

**字符串是不可变的：**
```python
s = "hello"
# s[0] = 'H'  # TypeError! 字符串不可变
s = "H" + s[1:]  # 只能创建新字符串 -> "Hello"
```

**字符串格式化（三种方式）：**
```python
name, age = "Alice", 30

# 1. % 格式化（旧式）
"名字: %s, 年龄: %d" % (name, age)

# 2. str.format()
"名字: {}, 年龄: {}".format(name, age)
"名字: {0}, 年龄: {1}".format(name, age)

# 3. f-string（推荐，Python 3.6+）
f"名字: {name}, 年龄: {age}"
f"计算: {2 ** 10}"          # 支持表达式
f"{'hello':>10}"            # 格式控制：右对齐，宽度10
f"{3.14159:.2f}"            # 保留2位小数 -> '3.14'
```
