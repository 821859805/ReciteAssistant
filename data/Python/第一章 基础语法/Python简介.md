Python 是一种高级、解释型、通用的编程语言，由 Guido van Rossum 于 1991 年发布。

**核心特点：**
- **解释型语言**：代码逐行解释执行，无需编译，开发效率高
- **动态类型**：变量无需声明类型，运行时自动推断
- **强类型**：不同类型之间不会隐式转换（如 `"3" + 5` 会报错）
- **缩进敏感**：使用缩进（通常4个空格）表示代码块，而非花括号
- **多范式**：支持面向对象、函数式、过程式编程

**Python 的设计哲学（The Zen of Python）：**
```python
import this
# 核心理念：
# Beautiful is better than ugly.（优美胜于丑陋）
# Explicit is better than implicit.（显式胜于隐式）
# Simple is better than complex.（简单胜于复杂）
# Readability counts.（可读性很重要）
```

**Python 2 vs Python 3：**
- Python 2 已于 2020 年停止维护
- Python 3 是当前主流版本，不完全向后兼容
- 主要区别：print 变为函数、默认 Unicode、整数除法行为等
