**显式类型转换：**
```python
# 转整数
int("42")          # 42
int(3.9)           # 3（截断，不是四舍五入）
int("0xff", 16)    # 255（指定进制）
int(True)          # 1

# 转浮点数
float("3.14")      # 3.14
float("inf")       # inf
float(42)          # 42.0

# 转字符串
str(42)            # "42"
str([1, 2, 3])     # "[1, 2, 3]"
repr("hello")      # "'hello'"（带引号的表示形式）

# 转布尔
bool(1)            # True
bool(0)            # False
bool("hello")      # True
bool("")           # False

# 转列表/元组/集合
list("abc")        # ['a', 'b', 'c']
list((1, 2, 3))    # [1, 2, 3]
tuple([1, 2, 3])   # (1, 2, 3)
set([1, 2, 2, 3])  # {1, 2, 3}
```

**str() 与 repr() 的区别：**
```python
s = "hello\nworld"
print(str(s))    # hello
                 # world  （面向用户的友好输出）
print(repr(s))   # 'hello\nworld' （面向开发者的精确表示）
```

**隐式类型转换：**
Python 在数值运算中会自动进行隐式转换：
```python
1 + 2.0        # 3.0（int -> float）
True + 1       # 2（bool -> int）
1 + 2j         # (1+2j)（int -> complex）
```
