**可迭代对象（Iterable）：**
实现了 `__iter__()` 方法的对象，可以用 `for` 循环遍历。
```python
# 内置可迭代对象
list, tuple, str, dict, set, range, file, ...

# 判断是否可迭代
from collections.abc import Iterable, Iterator
isinstance([1, 2], Iterable)   # True
isinstance(42, Iterable)       # False
```

**迭代器（Iterator）：**
实现了 `__iter__()` 和 `__next__()` 方法的对象。
```python
# 从可迭代对象获取迭代器
lst = [1, 2, 3]
it = iter(lst)           # 等价于 lst.__iter__()

next(it)  # 1             等价于 it.__next__()
next(it)  # 2
next(it)  # 3
next(it)  # StopIteration 异常

# for 循环的本质
# for x in iterable:
#     ...
# 等价于:
it = iter(iterable)
while True:
    try:
        x = next(it)
        # ...
    except StopIteration:
        break
```

**自定义迭代器：**
```python
class Countdown:
    def __init__(self, start):
        self.start = start
    
    def __iter__(self):
        return self  # 迭代器返回自身
    
    def __next__(self):
        if self.start <= 0:
            raise StopIteration
        self.start -= 1
        return self.start + 1

for n in Countdown(5):
    print(n)  # 5 4 3 2 1
```

**迭代器 vs 可迭代对象：**
| 特性 | 可迭代对象 | 迭代器 |
|------|-----------|--------|
| 方法 | `__iter__` | `__iter__` + `__next__` |
| 可重复遍历 | 是 | 否（耗尽后不可复用） |
| 例子 | list, str | iter(list), file object |

```python
# 迭代器一旦耗尽就无法重用
it = iter([1, 2, 3])
list(it)  # [1, 2, 3]
list(it)  # []（已耗尽）
```
