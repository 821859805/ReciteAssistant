**打开与关闭文件：**
```python
# 基本方式（需要手动关闭）
f = open("file.txt", "r", encoding="utf-8")
content = f.read()
f.close()

# 推荐方式：with 语句（自动关闭）
with open("file.txt", "r", encoding="utf-8") as f:
    content = f.read()
# 退出 with 块后文件自动关闭
```

**打开模式：**
| 模式 | 说明 |
|------|------|
| `r` | 只读（默认），文件不存在报错 |
| `w` | 写入，文件不存在则创建，存在则清空 |
| `a` | 追加，文件不存在则创建 |
| `x` | 独占创建，文件已存在报错 |
| `r+` | 读写 |
| `w+` | 读写（清空文件） |
| `b` | 二进制模式（如 `rb`, `wb`） |

**读取文件：**
```python
with open("file.txt", "r", encoding="utf-8") as f:
    # 方式1：读取全部内容
    content = f.read()
    
    # 方式2：读取一行
    line = f.readline()
    
    # 方式3：读取所有行（返回列表）
    lines = f.readlines()
    
    # 方式4：逐行迭代（推荐，内存友好）
    for line in f:
        print(line.strip())
    
    # 方式5：读取指定字节数
    chunk = f.read(1024)
```

**写入文件：**
```python
with open("output.txt", "w", encoding="utf-8") as f:
    f.write("第一行\n")
    f.write("第二行\n")
    
    # 写入多行
    lines = ["行1\n", "行2\n", "行3\n"]
    f.writelines(lines)

# 追加模式
with open("output.txt", "a", encoding="utf-8") as f:
    f.write("追加的内容\n")
```

**文件指针操作：**
```python
with open("file.txt", "r") as f:
    f.tell()         # 返回当前指针位置
    f.seek(0)        # 移到文件开头
    f.seek(10)       # 移到第10个字节
    f.seek(0, 2)     # 移到文件末尾（0=开头, 1=当前, 2=末尾）
```
