**包**是包含 `__init__.py` 的目录，用于组织相关模块。

**包的目录结构：**
```
mypackage/
    __init__.py          # 包的初始化文件（可以为空）
    module_a.py
    module_b.py
    subpackage/
        __init__.py
        module_c.py
```

**使用包：**
```python
# 导入模块
from mypackage import module_a
from mypackage.subpackage import module_c

# 导入模块中的函数
from mypackage.module_a import my_function
```

**__init__.py 的作用：**
```python
# mypackage/__init__.py

# 1. 控制 from package import * 的行为
__all__ = ["module_a", "module_b"]

# 2. 包级别的初始化代码
print("mypackage 被加载")

# 3. 提供便捷的导入路径
from .module_a import ClassA
from .module_b import ClassB
# 使用者可以直接: from mypackage import ClassA
```

**相对导入与绝对导入：**
```python
# 绝对导入（推荐）
from mypackage.module_a import func_a

# 相对导入（在包内部使用）
from . import module_b          # 从当前包导入
from .module_b import func_b    # 从当前包的模块导入
from .. import other_package    # 从上级包导入
from ..other import module      # 从上级包的子包导入
```

**pip 与虚拟环境：**
```bash
# 安装第三方包
pip install requests
pip install requests==2.28.0    # 指定版本
pip install -r requirements.txt # 从文件安装

# 虚拟环境
python -m venv myenv            # 创建虚拟环境
source myenv/bin/activate       # Linux/Mac 激活
myenv\Scripts\activate           # Windows 激活
deactivate                      # 退出虚拟环境

# 导出/导入依赖
pip freeze > requirements.txt
pip install -r requirements.txt
```
