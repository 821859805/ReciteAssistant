**模块**是一个 Python 文件（.py），包含函数、类、变量等。

**导入方式：**
```python
# 1. 导入整个模块
import math
math.sqrt(16)         # 4.0

# 2. 导入特定成员
from math import sqrt, pi
sqrt(16)              # 4.0（无需前缀）

# 3. 导入并重命名
import numpy as np
from datetime import datetime as dt

# 4. 导入所有成员（不推荐，可能命名冲突）
from math import *

# 5. 条件导入
try:
    import ujson as json
except ImportError:
    import json
```

**模块搜索路径：**
Python 按以下顺序搜索模块：
1. 当前目录
2. PYTHONPATH 环境变量指定的目录
3. 标准库目录
4. 第三方包目录（site-packages）

```python
import sys
print(sys.path)       # 查看搜索路径列表
sys.path.append("/my/custom/path")  # 添加自定义路径
```

**__name__ 的作用：**
```python
# mymodule.py
def main():
    print("主程序逻辑")

if __name__ == "__main__":
    # 仅当直接运行此文件时执行
    # 被 import 时不执行
    main()
```

**dir() 查看模块内容：**
```python
import math
dir(math)             # 列出 math 模块的所有属性
help(math.sqrt)       # 查看函数帮助
```
