**os 模块——操作系统交互：**
```python
import os
os.getcwd()                  # 当前工作目录
os.listdir(".")              # 列出目录内容
os.makedirs("a/b/c", exist_ok=True)  # 递归创建目录
os.path.join("a", "b", "c") # 'a/b/c'（路径拼接）
os.path.exists("file.txt")  # 判断路径是否存在
os.environ.get("HOME")      # 获取环境变量
```

**sys 模块——解释器相关：**
```python
import sys
sys.argv           # 命令行参数列表
sys.path           # 模块搜索路径
sys.version        # Python 版本
sys.exit(0)        # 退出程序
sys.stdin / sys.stdout / sys.stderr  # 标准 IO
```

**datetime 模块——日期时间：**
```python
from datetime import datetime, timedelta, date

now = datetime.now()
today = date.today()
dt = datetime(2024, 6, 15, 10, 30)
dt.strftime("%Y-%m-%d %H:%M")    # 格式化 -> "2024-06-15 10:30"
datetime.strptime("2024-06-15", "%Y-%m-%d")  # 解析
tomorrow = today + timedelta(days=1)
```

**collections 模块——高级数据结构：**
```python
from collections import Counter, defaultdict, deque, namedtuple, OrderedDict

Counter("abracadabra")           # Counter({'a': 5, 'b': 2, ...})
Counter("abc").most_common(2)    # [('a', 1), ('b', 1)]

dd = defaultdict(int)            # 默认值为 0
dd["missing"] += 1               # 1（不会 KeyError）

dq = deque([1, 2, 3])
dq.appendleft(0)                 # deque([0, 1, 2, 3])
dq.popleft()                     # 0（O(1)）
```

**json 模块——JSON 处理：**
```python
import json
data = {"name": "Alice", "age": 25}
json_str = json.dumps(data, ensure_ascii=False, indent=2)
parsed = json.loads(json_str)
```

**pathlib 模块——面向对象的路径处理（推荐）：**
```python
from pathlib import Path
p = Path("data") / "file.txt"    # 路径拼接
p.exists()                        # 是否存在
p.read_text()                     # 读取文本
p.write_text("hello")             # 写入文本
p.suffix                          # '.txt'
p.stem                            # 'file'
list(Path(".").glob("*.py"))      # 查找文件
```
