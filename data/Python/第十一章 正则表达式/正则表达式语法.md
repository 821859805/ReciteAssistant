**字符匹配：**
| 模式 | 说明 |
|------|------|
| `.` | 匹配任意字符（除换行符） |
| `\d` | 数字 [0-9] |
| `\D` | 非数字 |
| `\w` | 单词字符 [a-zA-Z0-9_] |
| `\W` | 非单词字符 |
| `\s` | 空白字符（空格、制表符、换行等） |
| `\S` | 非空白字符 |
| `[abc]` | 字符集合：a 或 b 或 c |
| `[^abc]` | 排除集合：除 a, b, c 外 |
| `[a-z]` | 范围：a 到 z |

**量词：**
| 模式 | 说明 |
|------|------|
| `*` | 0次或多次（贪婪） |
| `+` | 1次或多次（贪婪） |
| `?` | 0次或1次（贪婪） |
| `{n}` | 恰好n次 |
| `{n,m}` | n到m次 |
| `*?` | 0次或多次（非贪婪） |
| `+?` | 1次或多次（非贪婪） |

**锚点：**
| 模式 | 说明 |
|------|------|
| `^` | 字符串/行开头 |
| `$` | 字符串/行结尾 |
| `\b` | 单词边界 |
| `\B` | 非单词边界 |

**分组与引用：**
```python
# 捕获组
m = re.search(r"(\d{4})-(\d{2})-(\d{2})", "日期: 2024-06-15")
m.group(1)  # '2024'

# 命名组
m = re.search(r"(?P<year>\d{4})-(?P<month>\d{2})", "2024-06")
m.group("year")    # '2024'
m.groupdict()      # {'year': '2024', 'month': '06'}

# 非捕获组
re.findall(r"(?:http|https)://\S+", text)

# 反向引用
re.search(r"(\w+)\s+\1", "hello hello")  # 匹配重复单词

# 前瞻与后顾
re.findall(r"\w+(?=@)", "alice@test.com")  # ['alice']（正向前瞻）
re.findall(r"(?<=@)\w+", "alice@test.com") # ['test']（正向后顾）
```

**常用正则示例：**
```python
# 邮箱
r"[\w.+-]+@[\w-]+\.[\w.]+"

# 手机号（中国大陆）
r"1[3-9]\d{9}"

# IP 地址（简单版）
r"\d{1,3}(\.\d{1,3}){3}"

# URL
r"https?://[\w.-]+(?:/[\w./?%&=-]*)?"
```
