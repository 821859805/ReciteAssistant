**多态**是指不同类的对象对同一方法有不同的实现。

**传统多态（基于继承）：**
```python
class Shape:
    def area(self):
        raise NotImplementedError

class Circle(Shape):
    def __init__(self, radius):
        self.radius = radius
    def area(self):
        return 3.14159 * self.radius ** 2

class Rectangle(Shape):
    def __init__(self, width, height):
        self.width = width
        self.height = height
    def area(self):
        return self.width * self.height

# 统一接口
def print_area(shape):
    print(f"面积: {shape.area()}")

print_area(Circle(5))        # 面积: 78.53975
print_area(Rectangle(3, 4))  # 面积: 12
```

**鸭子类型（Duck Typing）：**
"如果它走起来像鸭子、叫起来像鸭子，那它就是鸭子。"
Python 不关心对象的类型，只关心它是否有所需的方法/属性。

```python
class Duck:
    def quack(self):
        print("嘎嘎！")

class Person:
    def quack(self):
        print("我在模仿鸭子叫！")

# 不检查类型，只要有 quack 方法就行
def make_it_quack(thing):
    thing.quack()

make_it_quack(Duck())     # 嘎嘎！
make_it_quack(Person())   # 我在模仿鸭子叫！
```

**协议（Protocol）——Python 内置的鸭子类型：**
```python
# 可迭代协议：实现 __iter__ 和 __next__
# 上下文管理器协议：实现 __enter__ 和 __exit__
# 可调用协议：实现 __call__

class MyRange:
    def __init__(self, n):
        self.n = n
        self.current = 0
    
    def __iter__(self):
        return self
    
    def __next__(self):
        if self.current >= self.n:
            raise StopIteration
        value = self.current
        self.current += 1
        return value

# 可以用在 for 循环中
for i in MyRange(5):
    print(i)  # 0, 1, 2, 3, 4
```

**typing.Protocol（Python 3.8+，显式协议）：**
```python
from typing import Protocol

class Drawable(Protocol):
    def draw(self) -> None: ...

def render(obj: Drawable) -> None:
    obj.draw()
# 任何有 draw() 方法的对象都可以传入，无需继承 Drawable
```
