**魔术方法（Magic Methods / Dunder Methods）** 是 Python 中以双下划线开头和结尾的特殊方法，用于定义对象的内置行为。

**对象表示：**
```python
class Point:
    def __init__(self, x, y):
        self.x = x
        self.y = y
    
    def __str__(self):
        """面向用户的字符串表示（print 时调用）"""
        return f"({self.x}, {self.y})"
    
    def __repr__(self):
        """面向开发者的字符串表示（调试用）"""
        return f"Point({self.x}, {self.y})"

p = Point(3, 4)
print(p)        # (3, 4)  —— 调用 __str__
repr(p)         # 'Point(3, 4)'  —— 调用 __repr__
```

**运算符重载：**
```python
class Vector:
    def __init__(self, x, y):
        self.x = x
        self.y = y
    
    def __add__(self, other):     # +
        return Vector(self.x + other.x, self.y + other.y)
    
    def __sub__(self, other):     # -
        return Vector(self.x - other.x, self.y - other.y)
    
    def __mul__(self, scalar):    # *
        return Vector(self.x * scalar, self.y * scalar)
    
    def __eq__(self, other):      # ==
        return self.x == other.x and self.y == other.y
    
    def __lt__(self, other):      # <
        return abs(self) < abs(other)
    
    def __abs__(self):            # abs()
        return (self.x ** 2 + self.y ** 2) ** 0.5
    
    def __len__(self):            # len()
        return 2
    
    def __bool__(self):           # bool()
        return self.x != 0 or self.y != 0

v1 = Vector(1, 2)
v2 = Vector(3, 4)
v3 = v1 + v2         # Vector(4, 6)
```

**容器相关：**
```python
class MyList:
    def __init__(self, data):
        self._data = list(data)
    
    def __getitem__(self, index):    # obj[key]
        return self._data[index]
    
    def __setitem__(self, index, value):  # obj[key] = value
        self._data[index] = value
    
    def __contains__(self, item):    # item in obj
        return item in self._data
    
    def __iter__(self):              # for x in obj
        return iter(self._data)
    
    def __len__(self):               # len(obj)
        return len(self._data)
```

**常用魔术方法一览：**
| 方法 | 触发方式 |
|------|----------|
| `__init__` | 初始化 |
| `__del__` | 对象销毁 |
| `__str__` | str() / print() |
| `__repr__` | repr() |
| `__hash__` | hash() |
| `__call__` | obj() |
| `__enter__/__exit__` | with 语句 |
