**类的定义：**
```python
class Dog:
    # 类属性（所有实例共享）
    species = "犬科"
    
    # 初始化方法（构造函数）
    def __init__(self, name, age):
        # 实例属性（每个实例独有）
        self.name = name
        self.age = age
    
    # 实例方法
    def bark(self):
        return f"{self.name} 汪汪叫！"
    
    def info(self):
        return f"{self.name}, {self.age}岁"
```

**创建和使用对象：**
```python
dog1 = Dog("旺财", 3)
dog2 = Dog("来福", 5)

print(dog1.bark())      # 旺财 汪汪叫！
print(dog2.info())      # 来福, 5岁
print(dog1.species)     # 犬科
print(Dog.species)      # 犬科（通过类访问）

# 动态添加属性
dog1.color = "棕色"     # 仅 dog1 有此属性
```

**self 的含义：**
- `self` 是对实例自身的引用，名字是约定（可以换但不推荐）
- 调用 `dog1.bark()` 时，Python 自动将 `dog1` 作为 `self` 传入
- 等价于 `Dog.bark(dog1)`

**类属性 vs 实例属性：**
```python
class Counter:
    count = 0                # 类属性
    
    def __init__(self):
        Counter.count += 1   # 通过类名修改类属性
        self.id = Counter.count  # 实例属性

c1 = Counter()
c2 = Counter()
print(Counter.count)  # 2
print(c1.id, c2.id)   # 1, 2
```
