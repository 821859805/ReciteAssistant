```python
class MyClass:
    class_var = "类变量"
    
    def __init__(self, value):
        self.value = value
    
    # 实例方法：第一个参数是 self（实例）
    def instance_method(self):
        return f"实例方法: value={self.value}, var={self.class_var}"
    
    # 类方法：第一个参数是 cls（类本身）
    @classmethod
    def class_method(cls):
        return f"类方法: var={cls.class_var}"
    
    # 静态方法：没有 self 或 cls，与普通函数类似
    @staticmethod
    def static_method():
        return "静态方法: 不访问类或实例属性"
```

**调用方式：**
```python
obj = MyClass("hello")

# 实例方法（通过实例调用）
obj.instance_method()

# 类方法（通过类或实例调用）
MyClass.class_method()
obj.class_method()

# 静态方法（通过类或实例调用）
MyClass.static_method()
obj.static_method()
```

**类方法的常见用途——工厂方法：**
```python
class Date:
    def __init__(self, year, month, day):
        self.year = year
        self.month = month
        self.day = day
    
    @classmethod
    def from_string(cls, date_str):
        """从字符串创建日期对象"""
        year, month, day = map(int, date_str.split("-"))
        return cls(year, month, day)
    
    @classmethod
    def today(cls):
        """获取今天的日期"""
        import datetime
        t = datetime.date.today()
        return cls(t.year, t.month, t.day)

d1 = Date(2024, 1, 1)
d2 = Date.from_string("2024-06-15")
d3 = Date.today()
```

**选择指南：**
| 方法类型 | 访问实例属性 | 访问类属性 | 典型用途 |
|----------|:---:|:---:|------|
| 实例方法 | ✅ | ✅ | 操作实例数据 |
| 类方法 | ❌ | ✅ | 工厂方法、操作类状态 |
| 静态方法 | ❌ | ❌ | 工具函数、逻辑分组 |
