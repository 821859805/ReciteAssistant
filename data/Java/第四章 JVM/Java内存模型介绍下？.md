Java内存由：虚拟机栈、本地方法栈、堆、元空间、程序计数器。五大部分组成

虚拟机栈：线程私有的内存空间，方法执行时会创建栈帧，栈帧包含：局部变量表、操作数栈、动态链接、方法出口。

本地方法栈：和虚拟机栈类似，只不过是为Native本地方法服务

堆：线程共享的一块内存空间，Java堆由年轻代和老年代组成，年轻代分为Eden园区、Survivor0和Survivor1区

元空间：JDK1.8以前，叫做方法区，用永久代实现，JDK1.8以后，移除了永久代，用元空间取代，存储在操作系统直接内存中，元空间存储：类信息、运行时常量池、静态变量

元空间补充：字符串常量池在JDK1.8之前存储在永久代中，JDK1.8之后存储到堆中

程序计数器：存储了当前线程正在执行的字节码指令的地址
